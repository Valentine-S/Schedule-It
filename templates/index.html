Overview

This is a simple web app that integrates with google calendar. You can use it to show you blocks of free time and schedule appointments.

Future Implementations

Schedule IT can be implemented in a Social Media app to create constructive competition between friends. Using the google calendar tasks as personal goals. Friends can compete to see who can complete the most personal goals.

Schedule IT can also be used as a time management solution. Keeping track of events and activities easier, it facilitates an increase in personal or business productivity. Google Calendar is a powerful tool that allows events to be shared via E-mail. This has a multitude of applications in any environment.
